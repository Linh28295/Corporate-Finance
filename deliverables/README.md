# deliverables/

Final, presentation-ready outputs. Only polished, reviewed work lands here — everything in this directory should be something you would confidently hand to a senior executive.

## What belongs here

- **Stage 5 raw LLM output** — the unedited response from executing the Stage 4 spec
- **Stage 5 evaluated final analysis** — edited, annotated, and corrected version with your own judgement applied
- **Any other final outputs** — polished summaries, presentation materials

## Naming convention

```
nguyen-stage5-KO-llm-raw.md         # Unedited LLM response
nguyen-stage5-KO-final-analysis.md  # Evaluated final version
```

## What does NOT belong here

- Working drafts — keep those in `docs/` or `analysis/`
- Source data — that belongs in `data/`
- Excel models — those belong in `models/builds/`
