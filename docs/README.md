# docs/

Written deliverables and reference materials for BUS 629: International Corporate Finance.

## Structure

| Subdirectory | Contents |
|--------------|----------|
| `decisions/` | Executive decision memos and analytical rationale (Stage 2) |
| `specs/` | Technical specifications for LLM-assisted analysis (Stage 4) |
| `plans/` | Optional project plans and timelines |
| `templates/` | Reusable document templates (memo, spec, naming conventions) |

## Naming Conventions

All written documents follow the format: `YYYY-MM-DD-{descriptive-slug}.md`

Example: `2026-05-10-manulife-company-selection-memo.md`
