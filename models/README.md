# models/

Financial models for BUS 629: International Corporate Finance.

## Structure

| Subdirectory | Contents |
|--------------|----------|
| `templates/` | Blank model frameworks — the instructor-provided ratios template lives here (Stage 1) |
| `builds/` | Populated, working models with real company financials (Stage 3) |

## Conventions

- All models are Excel (`.xlsx`) files.
- Templates are unmodified instructor-provided frameworks.
- Builds are named `{company-ticker}-ratios-{YYYY-MM-DD}.xlsx`.
- Never hardcode numbers in formula cells — use named ranges from the template.
